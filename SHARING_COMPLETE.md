A: Only if you choose Option 2 (Private Collaborators). Options 1 and 3 don't require accounts.

**Q: Can team members make changes?**  
A: Only if you give them Write/Admin access. Read access is view-only.

**Q: How do I update the shared code?**  
A: Commit and push changes. Team members pull updates (Git) or you share a new ZIP.

**Q: What if someone leaves the team?**  
A: Remove them from collaborators if using GitHub. ZIP files they have remain.

**Q: Is the code secure to share?**  
A: Yes - it's a client-side web app with no backend secrets. Review for any internal URLs first.

**Q: Can the app work offline?**
